Amazon Fullstack Clone

Full-stack Amazon clone built with Next.js, Tailwind CSS, and TypeScript.
This project focuses on modern UI/UX, clean architecture, and scalable frontend structure.

🚀 Tech Stack
Next.js (App Router)
TypeScript
Tailwind CSS
ESLint
📦 Getting Started

Run the development server:

npm run dev

Open http://localhost:3000 in your browser.

📁 Project Structure
src/
 ├── app/
 ├── components/
 ├── features/
 ├── lib/
